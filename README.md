# kicad-libs

Repositorio personal de librerías para KiCad 9:
- **symbols/**: librerías `.kicad_sym`
- **footprints/**: carpetas `.pretty` con `.kicad_mod`
- **3dmodels/**: modelos STEP/WRL organizados en `*.3dshapes`
- **docs/**: imágenes/notes de referencia

## Uso en KiCad
KiCad → Preferences → Manage Symbol Libraries / Footprint Libraries → Add existing...
